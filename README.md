# Practical Statistics for Data Scientists (Python)

## Purpose
This repository documents my chapter-by-chapter journey through *Practical Statistics for Data Scientists (2nd/3rd Edition)* by Peter Bruce, Andrew Bruce, and Peter Gedeck. It serves as a personal knowledge base and a practical implementation of the book's concepts exclusively in Python.

## Goal
To build a strong statistical foundation for Data Science and Machine Learning Engineering by translating theoretical concepts into executable Python code, solving book exercises, and running personal experiments.
